QR kód pro festival - GitHub verze

Informace o GitHub verzi:

Tato verze generuje QR kód, který odkazuje na tuto stránku s ID vašeho festivalu. Všechny obrázky jsou načítány z vašeho GitHub repozitáře.

Nastavení repozitáře

Pro správné zobrazení obrázků musíte nejdříve nastavit informace o svém GitHub repozitáři.

Instrukce pro nahrání obrázků na GitHub:

  1. Nahrajte obrázky do svého GitHub repozitáře
  2. Ujistěte se, že jsou obrázky ve formátu .jpg, .jpeg, .png nebo .gif
  3. Do formuláře níže zadejte pouze názvy souborů (např. "drinks.jpg")
  4. Systém automaticky vytvoří odkazy na "raw" verze obrázků

Stáhnout QR kód

Návod k použití:

  1. Nastavte informace o vašem GitHub repozitáři
  2. Vložte unikátní ID vašeho festivalu
  3. Zadejte názvy obrázků, které máte v GitHub repozitáři
  4. Klikněte na "Náhled" pro kontrolu zobrazení
  5. Klikněte na tlačítko "Uložit data a generovat QR kód"
  6. Stáhněte si vygenerovaný QR kód
  7. Vytiskněte QR kód a umístěte ho do propagačních materiálů

Informace o festivalu

Pokud jste sem přišli ze skenu QR kódu, informace by se měly automaticky zobrazit.

Pokud nevidíte žádné informace, kontaktujte organizátory festivalu.
